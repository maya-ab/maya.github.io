Flutter Puzzle Challenge - Space Slide Puzzle

This project was created to enter the flutter puzzle hack. I built upon the sample code given and created a space themed puzzle My goal was to make the classic slide puzzle game more difficult and interesting for the user. This project was built entirely in flutter

  • When the puzzle is completed, the software executes an animation of our solar system

Helping Bees

This project was created to enter a Technovation competition. Our team came up with the idea for an app called Helping Bees, an app that helps people find volunteer opportunities easily. We won 1st place in the Calgary Regional Senior Division. After the competition was over our team of 4 collectively decided to continue working on it with hopes of releasing it. Currently, we are working on finishing up development on the app

I worked on developing/designing the log in, sign up, search, and map view. The design process was more so collaborative and the coding was done separately. After the UI was developed I also worked on integrating it with our database
  • As a volunteer you can search through volunteer posts, like posts to view them later, and select fields which you are interested in volunteering in
  • As a organization, you can add volunteer posts and view your uploaded posts on your profile page
  • This project was built using flutter, firebase, and google maps

Quick Yard Help

Quick Yard Help (QYH) was made to help homeowners or renters with yards to find someone to help them out with yard work. This project was mostly done for fun.
  • The home page consists of a map with dots indicating where users are located, a green dot is someone willing to provide yard work, a blue dot is someone looking for yard work help
  • Users can click the menu button to view and manage their profile
  • To find someone to provide yard work users simply click on a green dot and click request help and then they wait for a response
  • If the other user accepts the request you can then chat with them and see updates on their progress (This process is shown in the video below)
  • Users have the option of blocking other users
  • QYH was developed using swift and firebase
  • See Quick Yard Help on github

Movie and TV Show Tracker

This project was created for an assignment for my computer science 210 class. It allows you to track the movies and tv shows you've watched or want to watch.
  • If a user wants to record something they want to watch they, simply type the title, select if it's a movie/show, and click save
  • If it is a movie/show that a user has watched already they can additionally add a note and a rating
  • Users can also save their entries and load them up again after exiting the app
  • Upon clicking save a small pop window loads with a gif letting user know the save was successful
  • Made using java
  • See Movie and Show Tracker on github

GDP and CO2 Emissions Global Analysis

This project was created as part of a project for my computer science 301 class. We worked in teams but each had separate specific research topics. Here I'm displaying only my section of the research. The research areas we chose were CO2 emissions, GDP, and global forest area. I focused on CO2 emissions and GDP. My research questions were:

Over the years, how does the relationship between GDP and CO2 emissions compare for the 15 highest income Countries vs the 15 lowest income countries? What about during financial crises periods?

How does the average CO2 emissions per capita of high GDP countries compare to low GDP countries, and the rest of the world? Has this changed over the years?
  • All of the analysis was done in python
  • I utilized pandas, numpy, seaborn, and matlab libraries to create visualizations from my two datasets
  • Using these visualizations I answered my research questions (Only a portion of my visualizations are displayed here, the rest can be found on github)
  • See analysis on github
